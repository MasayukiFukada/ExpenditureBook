# Expenditure Book

* 支出を専門的に管理するための家計簿
* 総資産を把握するのが目的ではなく、日常における支出の金額やその傾向を俯瞰するのが目的

## 技術スタック

* バックエンド
  * Java + Spring Boot + Spring Data JPA
  * Docker
    * DB: MySQL
* フロントエンド
  * Vue.js (?) + billboard.js
  * Bulma CSS framework
